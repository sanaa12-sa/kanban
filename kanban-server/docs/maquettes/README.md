# Maquettes de l'application Kanban

## US-01 : Page de connexion / inscription
- Champs : Email, Mot de passe
- Boutons : Se connecter, S'inscrire

## US-02 : Tableau de bord (liste des boards)
- Liste des boards de l'utilisateur
- Bouton : Créer un nouveau board
- Chaque board affiche : titre, couleur, nombre de membres

## US-03 : Vue d'un Board (colonnes + tâches)
- 4 colonnes par défaut : À faire | En cours | En révision | Terminé
- Chaque colonne affiche : nom, compteur de tâches, limite WIP
- Chaque tâche affiche : titre, priorité (couleur), assigné à, deadline
- Drag & drop entre colonnes

## US-04 : Création / modification d'une tâche
- Champs : Titre, Description, Priorité, Assigné à, Deadline
- Section commentaires en bas

## US-05 : Gestion des membres
- Liste des membres du board
- Inviter par email
- Rôles : Admin / Membre
